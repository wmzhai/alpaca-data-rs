# Examples

This document describes the public example surface for `alpaca-data`.

## Current Task Boundary

The repository is currently in `Phase 6: Release Hardening`.

Runnable examples under `examples/` are added in the next task of this phase. This page is created first so the public docs surface already has a stable place for example guidance.

## Example Style

Examples in this repository follow these rules:

- async-only
- minimal happy-path code
- official method names
- no extra abstraction layer
- environment-driven authentication
